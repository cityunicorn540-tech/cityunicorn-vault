We make html instant browsers games for android in collab with Gemini AI. We have 2 methods. 1. Html code. 2. WASM-4 bundled html index.html and we change things in wasm4 like removing the 4 wasm4 colours and adding our own 4 colours. Also we chose not to keep the 64kb rule for wasm4 games.
Built with AssemblyScript for the WASM-4 fantasy console but converted to html instant browser games for android. We used termux android.
 Cityunicorn540-tech in collab with Gemini AI told WASM-4 to use AssemblyScript. w4 new neon-pong --as. Then we wanted an index.html game file. w4 bundle build/cart.wasm --html index.html --title "Neon Pong" We changed the 4 wasm4 colours to our desired 4 colours.
A list of WASM-4 games we made for the fantasy console but converted to html instant browsers games with our own 4 colours.
1. Neon-Pong.A multiplayer Pong game.
